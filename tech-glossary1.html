Tech Glossary

16 Essential Terms — The more beginner you are, the more this helps 🚀

AI / LLM
RAG
Retrieval-Augmented Generation
A technique that supercharges AI by letting it search a knowledge base before answering. Instead of relying only on training data, the model retrieves fresh relevant documents and uses them to generate more accurate responses.
Example: A support bot queries your internal docs in real-time, then answers based on your actual product manuals — not guesses.
🧠
Business / Finance
ROI
Return on Investment
A metric that measures how much value you get back relative to what you put in. Formula: (Gain − Cost) ÷ Cost × 100. Positive ROI = paid off. Negative ROI = loss.
Example: Spent $10,000 on ads → earned $30,000 in revenue → ROI = 200%.
📈
Sales / Software
CRM
Customer Relationship Management
A system for managing all interactions with current and potential customers. Centralizes contacts, tracks deals, logs conversations, and ensures no lead falls through the cracks.
Example: Airtable, HubSpot, Salesforce — log every touchpoint from first contact to closed deal.
🤝
Web / Mobile
PWA
Progressive Web App
A website that behaves like a native mobile app — installable on your home screen, works offline, and sends push notifications. Built with standard web tech but feels like a real app with no App Store needed.
Example: Twitter Lite, Starbucks, Pinterest — all installed directly from the browser.
📱
AI / Databases
VectorDB
Vector Database
A database that stores and searches by meaning, not keywords. It converts text, images, or audio into numbers and finds results based on semantic similarity — how close the meaning is, not exact word matches.
Example: Pinecone, Weaviate, Qdrant — the backbone of most RAG systems.
🗄️
Marketing / Sales
Funnel
Sales / Marketing Funnel
A model of the customer journey from first hearing about you to buying. It narrows at each stage: Awareness → Interest → Consideration → Purchase → Loyalty. Many people enter, few convert at the bottom.
Example: 10,000 see an ad → 1,000 sign up → 100 buy → 20 become loyal customers.
🔻
Engineering / Sales
Pipeline
Data / Sales Pipeline
A structured sequence of stages that data or deals flow through. In sales it tracks deals. In AI/data it's the automated chain that moves, cleans, and delivers data from source to destination.
Sales: Lead → Qualified → Proposal → Closed.
Data: Ingest → Clean → Transform → Serve.
⚙️
AI / Core
LLM
Large Language Model
The AI brain behind ChatGPT, Claude, and Gemini. Trained on massive text data, it predicts the most likely next word — so well it can write, reason, code, and converse. It's stateless: it remembers nothing between sessions on its own.
Example: GPT-4, Claude, Gemini, Llama — all LLMs running under the hood of AI products.
🤖
AI / LLM
Context Window
The LLM's Working Memory
The maximum amount of text an LLM can read and consider at one time. Your instructions, documents, and chat history all must fit inside it. Once full, older info gets dropped. Bigger window = more the AI can hold in mind at once.
Example: Claude has up to 1M tokens (~700,000 words) of context. GPT-4 has ~128k tokens.
🪟
AI / Prompting
System Prompt
Hidden AI Instructions
Secret instructions given to an AI before the conversation starts. Sets personality, rules, tone, and limits. The user never sees it — but it shapes every single response the AI gives throughout the session.
Example: "You are a helpful sales assistant. Never discuss competitors. Always reply in Arabic." — that's a system prompt.
🔧
AI / Core
Tokens
The Currency of LLMs
The tiny chunks an LLM reads and writes — not exactly words, more like word-pieces. Every model has a token limit per request and API pricing is based on tokens consumed. More tokens = higher cost and more context used.
Example: 1,000 tokens ≈ 750 words. A long document could be 50,000+ tokens.
🪙
AI / Agents
MCP
Model Context Protocol
A standard way for AI agents to connect to external tools and data sources — like a universal plug for AI. Instead of custom-coding every integration, MCP lets agents discover what resources are available and query them on the fly.
Example: An AI agent uses MCP to connect to Google Drive, your CRM, and a database — all through one standard interface.
🔌
AI / Optimization
Compaction
Context Compression
Shrinking large pieces of content into short summaries before passing them to an AI — to save space in the context window. Use one LLM call to summarize, then pass the compact version to the next step instead of the full raw document.
Example: A 50,000-token report → summarized into 500 words → passed to the next agent step cleanly.
🗜️
AI / Automation
Agent
Agentic AI / AI Agent
An AI that doesn't just answer questions — it takes actions. It can call tools, search databases, send messages, make decisions, and loop through tasks on its own until a goal is complete. Think AI employee, not just a chatbot.
Example: Your Instagram DM bot — reads messages, checks CRM, uploads to Drive, posts to Instagram. That's an agent.
🦾
Automation / Dev
Cron
Cron Job / Scheduled Task
A time-based scheduler that automatically triggers a task at a set time — every hour, every Monday, every month — with zero human input. Like setting an alarm, but instead of waking you up, it runs a workflow automatically.
Example: Every Monday at 9am → DM all active sellers to check if items are still available. That's a Cron job in n8n.
⏰
Business / Software
SaaS
Software as a Service
Software you access through a browser and pay for monthly — no installation, no downloads, no IT setup. The company hosts everything. You just log in from anywhere. Most tools in the modern AI automation stack are SaaS products.
Example: Notion, Airtable, Canva, Gmail, n8n Cloud — all SaaS. You connect them, you don't host them.
